1.分组
当多个元素的样式相同的时候，我们没有必要重复地为每个元素都设置样式，我们可以通过在多个选择器之间使用逗号分隔的分组选择器来统一设置元素样式。
例如

color: red;

3.after

此时页面上所有的标签都会继承body的字体颜色。然而css继承性的权重是非常低的，是比普通元素权重还要低的0.

我们只要给对应的标签设置字体颜色就可覆盖掉他继承的样式

color: green;

}
